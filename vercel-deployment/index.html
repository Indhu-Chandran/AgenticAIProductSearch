Product Catalog Query Engine

Application Information

This is a product catalog query engine built with OpenAI embeddings and Streamlit.

The application is hosted on a separate platform as Streamlit apps cannot be directly deployed to Vercel.

Deployment Options

For Streamlit applications, we recommend the following deployment options:

  • Streamlit Cloud: The official platform for Streamlit apps
  • Heroku: A platform that supports Python applications
  • Render: A cloud provider that supports Streamlit
  • AWS, GCP, or Azure: Cloud platforms with support for Python web applications

Streamlit Cloud Deployment

To deploy this application to Streamlit Cloud:

  1. Push your code to a GitHub repository
  2. Sign up for Streamlit Cloud
  3. Connect your GitHub repository
  4. Configure your app settings
  5. Deploy the application
